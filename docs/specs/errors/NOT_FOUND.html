NOT_FOUND

HTTP 404 · Not Found

리소스를 찾을 수 없습니다.

의미

일반 리소스 없음 폴백. 가능하면 도메인별 코드(PRODUCT_NOT_FOUND 등)가 우선 사용됩니다. RFC 9457 application/problem+json 형식으로 전달됩니다.

발생 조건

  • 도메인 코드가 없는 경우의 폴백

응답 예시

HTTP/1.1 404 Not Found
Content-Type: application/problem+json
{
  "type": "https://docs.goods-mall.dev/errors/NOT_FOUND",
  "title": "Not found",
  "status": 404,
  "code": "NOT_FOUND",
  "detail": "리소스를 찾을 수 없습니다.",
  "instance": "/api/v1/..."
}

필드

필드의미
type이 페이지의 URI — 에러 종류의 안정적 식별자이자 문서 링크.
title종류별 고정 요약(영어·불변, 로그·관측 그룹핑용).
statusHTTP 상태 코드(404).
code클라이언트가 분기에 사용하는 안정 키. 메시지 대신 이 값으로 처리.
detail이번 발생 건의 사람용 메시지(한국어·가변).
instance에러가 난 요청 경로.

클라이언트 처리

detail을 파싱하지 말고 code로 분기하세요.

if (problem.code === 'NOT_FOUND') {
  // Common 에러 처리
}
