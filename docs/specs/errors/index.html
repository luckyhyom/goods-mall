에러 카탈로그

RFC 9457 Problem Details의 code별 문서. 에러 응답의 type URI가 각 페이지를 가리킵니다.

Common

HTTP 422 · Unprocessable EntityVALIDATION_ERROR요청 값이 올바르지 않습니다.HTTP 401 · UnauthorizedUNAUTHORIZED인증이 필요합니다.HTTP 403 · ForbiddenFORBIDDEN권한이 없습니다.HTTP 404 · Not FoundNOT_FOUND리소스를 찾을 수 없습니다.HTTP 500 · Internal Server ErrorINTERNAL_ERROR서버 오류가 발생했습니다.

Auth

HTTP 409 · ConflictAUTH_EMAIL_TAKEN이미 가입된 이메일입니다.HTTP 401 · UnauthorizedAUTH_INVALID_CREDENTIALS이메일 또는 비밀번호가 올바르지 않습니다.HTTP 401 · UnauthorizedAUTH_LOCAL_DISABLED소셜 로그인 전용 계정입니다.HTTP 401 · UnauthorizedAUTH_TOKEN_EXPIRED토큰이 만료되었습니다.HTTP 401 · UnauthorizedAUTH_REFRESH_INVALID유효하지 않은 refresh 토큰입니다.HTTP 401 · UnauthorizedAUTH_REFRESH_REUSED토큰 재사용이 감지되어 모든 세션이 무효화되었습니다.HTTP 401 · UnauthorizedAUTH_LINK_INVALID연결 정보가 올바르지 않습니다.

Product

HTTP 404 · Not FoundPRODUCT_NOT_FOUND상품을 찾을 수 없습니다.HTTP 409 · ConflictPRODUCT_OUT_OF_STOCK재고가 부족합니다.

Cart

HTTP 409 · ConflictCART_EMPTY카트가 비어 있어 주문할 수 없습니다.HTTP 404 · Not FoundCART_ITEM_NOT_FOUND카트에 해당 상품이 없습니다.

Address

HTTP 404 · Not FoundADDRESS_NOT_FOUND주소를 찾을 수 없습니다.

Order

HTTP 404 · Not FoundORDER_NOT_FOUND주문을 찾을 수 없습니다.HTTP 409 · ConflictORDER_NOT_PENDING처리할 수 있는 상태가 아닙니다.HTTP 409 · ConflictORDER_ALREADY_PAID이미 결제된 주문입니다.HTTP 409 · ConflictORDER_ALREADY_CANCELED이미 취소된 주문입니다.HTTP 400 · Bad RequestIDEMPOTENCY_KEY_REQUIREDIdempotency-Key 헤더가 필요합니다.HTTP 409 · ConflictIDEMPOTENCY_KEY_CONFLICT같은 Idempotency-Key로 다른 요청이 처리되었습니다.
